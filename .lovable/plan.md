

## Квиз-платформа

### Авторизация
- Регистрация и вход по email + пароль через Supabase Auth
- Страница сброса пароля
- Профили пользователей (имя, аватар)
- Роль «admin» через отдельную таблицу `user_roles`

### Каталог квизов (для всех пользователей)
- Список доступных квизов с карточками: название, описание, количество вопросов, категория
- Фильтрация/поиск по названию и категории

### Прохождение квиза
- Пошаговый интерфейс: один вопрос на экран с 2-4 вариантами ответа (один правильный)
- Прогресс-бар сверху
- По завершении — экран результата: счёт, правильные/неправильные ответы с разбором

### Статистика и история
- Сохранение каждой попытки в БД (пользователь, квиз, ответы, счёт, дата)
- Страница «Мои результаты» с историей прохождений и лучшим результатом по каждому квизу
- Лидерборд по каждому квизу (топ-10)
- Детальная аналитика: процент правильных ответов по вопросам

### Админ-панель (только для admin)
- Загрузка квиза из JSON-файла (формат: название, описание, массив вопросов с вариантами)
- Визуальный редактор: редактирование вопросов, вариантов, правильных ответов
- Публикация/скрытие квизов
- Просмотр общей статистики по квизам (сколько прошли, средний балл)

### База данных (Supabase)
- Таблицы: `profiles`, `user_roles`, `quizzes`, `questions`, `answers` (варианты), `quiz_attempts`, `attempt_answers`
- RLS-политики: пользователи видят только опубликованные квизы, админ — всё

### Дизайн
- Минималистичный, светлая тема, чистые карточки
- Анимации переходов между вопросами
